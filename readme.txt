Material Actions

Navigate the WordPress admin interface was never that beautiful and simple. Easily customize the links you use the most to be always on your side. This is also the perfect add-on to our other plugin, Material WP: http://codecanyon.net/item/material-wp-material-design-dashboard-theme/12981098?ref=732

== Installation ==

1. Upload 'material-actions' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
